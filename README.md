# ✅ Responsive Todo List App

A clean, modern, and fully responsive **Todo List** application built with **HTML5**, **CSS3**, and **JavaScript**.

Designed to help users organize their daily tasks through a simple, intuitive, and user-friendly interface while providing a smooth experience across desktop, tablet, and mobile devices.

---

## ✨ Features

- ➕ Add new tasks instantly
- 🗑️ Delete tasks with a single click
- ⚡ Dynamic interactions powered by JavaScript
- 📱 Fully Responsive Design
- 🎨 Clean & Minimal User Interface
- 💻 Optimized for Desktop, Tablet & Mobile
- 🧩 Semantic HTML5 Structure
- ✨ Well-Organized & Maintainable Code
